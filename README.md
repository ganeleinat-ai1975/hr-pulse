# HR Pulse — מערכת ניתוח שביעות רצון עובדים

> לוח בקרה אינטראקטיבי לניתוח סקרי עובדים, זיהוי סיכוני עזיבה, וניתוח ביצועי מנהלים

## גישה מהירה

**האפליקציה החיה:**  
[https://hr-pulse-7lhc9hqsoefettnf8mikhe.streamlit.app](https://hr-pulse-7lhc9hqsoefettnf8mikhe.streamlit.app)

פותחים בכל דפדפן — לא צריך להתקין כלום.

---

## מה המערכת עושה?

| מסך | תיאור |
|-----|--------|
| **תמונת מצב כוללת** | KPIים, ציון מחוברות, רמזור ארגוני, גרפים |
| **ניתוח מחלקות** | השוואת ציונים בין מחלקות, זיהוי פערים |
| **ניתוח מנהלים** | Top 5 / Bottom 5, מכתבי Coaching אישיים |
| **סיכון עזיבה** | Flight Risk Score (0–100) לכל עובד |
| **ניתוח תשובות פתוחות** | נושאים חוזרים, ציטוטים, sentiment |
| **המלצות אסטרטגיות** | מבוססות נתונים אמיתיים עם ROI משוער |
| **יצוא דוחות** | Word לדירקטוריון, Excel, מכתבי Coaching |
| **ניהול נתונים** | ייבוא Excel, עריכה, איפוס לדמו |

---

## נתוני הדגמה

המערכת מגיעה עם **1,010 עובדי דמו** ב-8 מחלקות:

| מחלקה | עובדים |
|--------|--------|
| פיתוח מוצר | 180 |
| מכירות | 200 |
| שירות לקוחות | 180 |
| פעילות | 150 |
| מחקר ופיתוח | 100 |
| שיווק | 90 |
| כספים | 70 |
| משאבי אנוש | 40 |

**ממצאים בנתוני הדגמה:**
- ציון מחוברות כולל: **3.07/5** (צהוב — דורש תשומת לב)
- 12.1% בסיכון עזיבה גבוה (122 עובדים)
- מנהלים "בעייתיים" מובנים לדמו של Coaching

---

## מודל סיכון עזיבה (Flight Risk Score)

ציון 0–100 לכל עובד, מחושב לפי:

| גורם | משקל |
|------|------|
| כוונת עזיבה (1–5) | 40% |
| eNPS | 20% |
| שביעות רצון משכר | 15% |
| שביעות רצון מפיתוח | 15% |
| שביעות רצון ממנהל | 10% |

**סף סיכון גבוה:** ≥65 נקודות

## הרצה מקומית

```bash
git clone https://github.com/ganeleinat-ai1975/hr-pulse
cd hr-pulse
python -m venv .venv
.venv/bin/pip install -r requirements.txt
.venv/bin/python generate_demo.py
.venv/bin/python -m streamlit run app.py
```

פותחים: [http://localhost:8501](http://localhost:8501)

## ייבוא נתונים אמיתיים

1. הכינו Excel עם עמודות: `שם, מחלקה, מנהל, שכר (1-5), פיתוח (1-5), מנהל (1-5), איזון (1-5), כוונת עזיבה (1-5), eNPS, משוב פתוח`
2. עברו למסך **ניהול נתונים**
3. לחצו **ייבוא מ-Excel** ומפו את העמודות
4. המערכת תחשב Flight Risk Score אוטומטית
